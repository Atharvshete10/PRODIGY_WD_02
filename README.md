# PRODIGY_WD_02 - Stopwatch Web Application

## 📌 Task Description

Build an interactive and user-friendly stopwatch web application using HTML, CSS, and JavaScript. The application allows users to accurately measure elapsed time with features such as start, pause, reset, and lap recording.

## 🚀 Features

- ⏱️ Start Stopwatch
- ⏸️ Pause Stopwatch
- 🔄 Reset Stopwatch
- 🏁 Record Multiple Lap Times
- 📱 Fully Responsive Design
- 🎨 Modern and Clean User Interface
- ⚡ Real-Time Time Updates

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## ⚙️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Atharvshete10/PRODIGY_WD_02.git
```

2. Open the project folder.

3. Open `index.html` using **Live Server** in VS Code or any modern web browser.
